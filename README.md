 # CoffeeApp
 How to install and use the web-project? 
 1) Copy this repository on your local machine with command: git clone https://github.com/Kyle-Butler01/CoffeeApp
 2) Go to file 'webstore/.env' and setting up your envirenments: database, debug, secret_key for you django project, cloudinary API_KEY, cloudinary API_SECRET, cloudinary CLOUD_NAME.
 for last 3 environments go regostration on https://cloudinary.com/ and take this on your cloudinary profile
 3) Lets check have you errors or some problems with it: 'python3 manage.py check' 
 4) start this project on your local server with command: 'python3 manage.py runserver'
 
 API documentation:
 all information about API -> route 'https://coffeeapp-376t.onrender.com/api/docs' on this site, if you use or test this site on localhost -> http://127.0.0.1:8000/api/docs
